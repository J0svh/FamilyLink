# FamilyLink - Agente Instructions

## Project Overview
FamilyLink es una aplicación de localización familiar con arquitectura hexagonal y DDD.

## Key Files
- `requirements.md` - Requisitos funcionales y no funcionales
- `design.md` - Diseño técnico completo
- `tasks.md` - Lista de tareas pendientes
- `specs/email-value-object.md` - Plan actual para Email.ts

## Tech Stack
- Backend: Node.js + TypeScript + Express
- Frontend: React + Vite + TypeScript
- Database: PostgreSQL
- Architecture: Hexagonal (Ports & Adapters)

## Active Agents
- Engram: Memoria persistente (comandos: `engram context`, `engram save`, `engram search`)
